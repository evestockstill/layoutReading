The LAYOUT refers to the placement of text, images, and other objects on a page. When a element has not been position in anyway it has NORMAL FLOW. Position RELATIVE moves an element relative to where it would have otherwise been, it does not affect elements that surround it, and an empty space is left where it was. When the position of an element is relative to its nearest positioned ancestor(first parent) and does not leave an empty space it is position: ABSOLUTE . Position: FIXED positions an element relative to the browser window and scrolls with the page. A layout that changes to accommodate for screen size is LIQUID layout.
